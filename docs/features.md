# HBC Features

## Implemented
* Login
* Comp listing
* User profiles


## Required to run a comp
* User roles
* Register a new comp
* Set comp styles
* Accept entries
* Register judges
* Register stewards
* Assign entries to tables
* Assign judges to tables
* Assign stewards to tables
* input scores
* Upload scoresheets
* Allow users to download scoresheets
* Show results
